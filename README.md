# Samurosaki-Python-Oto-Macro
"Samurosaki tarafından geliştirilen bu yeni otomasyon aracı, siz AFK iken otomatik tuş gönderimi yaparak işlerinizi kolaylaştırır. Giriş ekranı, Webhook desteği ve Admin paneli ile donatılan yazılımın, ilerleyen süreçte .exe formatına dönüştürülmesi ve uzak sunucudan otomatik güncelleme alması planlanmaktadır."
